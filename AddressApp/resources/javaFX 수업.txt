JavaFX 튜터리얼을 검색하던 중 이 사이트가 괜찮은 듯하여 
아래의 사이트 순서대로 따라하기 하면서 수업을 진행하겠습니다.

https://code.makery.ch/ 

위 사이트에서 
  Library  ->  JavaFX 
 : JavaFX Tutorial 에서 "한국어" 클릭 
   https://code.makery.ch/kr/library/javafx-tutorial/

해당 튜터리얼을 천천히 따라해 보시면 
기본적인 JavaFX 사용법을 익히기에 충분하다고 생각합니다.

1. 테이블 행을 클릭하면 회원정보를 보여준다. 

2. [신규] 버튼 클릭
    - 입력 영역 : 활성화, 입력박스 클리어
    - 버튼 상태 : [신규],[변경] 비활성, [저장], [취소] 활성화
    
3. [변경] 버튼 클릭
    - 입력 영역 : 활성화
    - 버튼 상태 : [신규], [변경] 비활성, [저장],[취소] 활성화 
    
4. [저장] 버튼 클릭
    저장여부 확인
    신규라면 : DB에 해당 회원이 존재하는지 확인
       존재 : 해당 아이디가 존재합니다. 
       존재하지 않으면 DB에 저장 (insert) 
		    -입력 영역 : 비활성화
		    -버튼 상태 : [신규],[변경] 활성화, [저장], [취소] 비활성화
     
    기존회원이면 
    DB에 저장(update)
	    -입력 영역 : 비활성화
		-버튼 상태 : [신규],[변경] 활성화, [저장], [취소] 비활성화
		
 5. [취소] 버튼 클릭
 	?? 입력영역에는 뭘 보여줄거니?? (신규에서, 변경에서)
 		-입력 영역 : 비활성화
 		-버튼 상태 : [신규],[변경] 활성화, [저장], [취소] 비활성화
